# Diamanten-Preisanalyse & Lineare Regression

Dieses Projekt umfasst eine vollständige statistische Analyse des diamonds-Datensatzes aus dem ggplot2-Paket. Ziel war es, den Preis von Diamanten basierend auf ihren physischen und qualitativen Eigenschaften zu verstehen und ihre Preise vorherzusagen.

## Projektübersicht

Das Projekt wurde als Jupyter Notebook (.ipynb) mit dem R-Kernel entwickelt und sowohl als R-File und HTML exportiert.

## Ausführung

Um den HTML-Report aus dem Notebook neu zu generieren, muss der folgenden Befehl im Terminal verwendet werden:

```bash
quarto render project_regression.ipynb --to html
```
